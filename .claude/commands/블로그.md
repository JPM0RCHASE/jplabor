# 스포티파이 형태로 블로그 자료 만들기

홈페이지의 JP 칼럼/가이드북/사례 내용을 받아 네이버 블로그용 이미지 3장을 생성합니다.

## 생성 구조
1. **썸네일** (1200×630) — 스포티파이 다크 톤, 제목 강조
2. **본문** — 섹션 헤더(초록) + 인용 박스 + 경고 박스 + 태그
3. **면책조항** — 밝은 톤, 블로그 하단 삽입용

## 실행 방법

사용자가 "스포티파이 형태로 블로그 자료 만들어줘 — [제목 또는 내용]" 라고 하면:

1. 홈페이지 `index.html`에서 해당 칼럼/가이드 내용을 읽어온다
2. `blog-assets/` 폴더에 `blog-[slug].html` 파일을 생성한다
   - 파일 구조: #thumb / #body / #disc 세 섹션
   - 디자인: 스포티파이 다크(#121212, #1ed760) + 밝은 본문
   - 본문 섹션 헤더: `.sec-head` (초록 좌측 포인트)
   - 인용 박스: `.quote` (연녹 배경)
   - 경고 박스: `.warn` (주황 배경)
3. `shot-blog.mjs`로 3개 PNG 자동 캡처:
   ```bash
   node blog-assets/shot-blog.mjs blog-assets/blog-[slug].html
   ```
4. 결과물 3장을 사용자에게 보여주고 파일 위치 안내

## 출력 파일
```
blog-assets/
  output-[slug]-thumb.png   ← 네이버 대표 이미지
  output-[slug]-body.png    ← 본문 (붙여넣기 후 이 이미지 삽입 or 텍스트 사용)
  output-[slug]-disc.png    ← 글 맨 아래 면책조항
```

## 네이버 블로그 업로드 순서
1. 글쓰기 → 대표 이미지: `thumb.png`
2. 본문에 이미지 삽입: `body.png`
3. 맨 아래 이미지 삽입: `disc.png`
4. 발행
