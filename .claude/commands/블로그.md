# 스포티파이 형태로 블로그 자료 만들기

홈페이지의 JP 칼럼/가이드북/사례 내용을 받아 네이버 블로그용 이미지 3장을 생성합니다.

## 생성 구조
1. **썸네일** (1200×630) — 스포티파이 다크 톤, 제목 강조 → **이미지(.png)**
2. **본문** — 섹션 구분 + 인용 → **텍스트(.txt)** (검색 색인 위해 이미지 아님!)
3. **면책조항** — 밝은 톤, 블로그 하단 삽입용 → **이미지(.png)**

⚠️ 본문은 반드시 텍스트로. 이미지로 만들면 네이버 검색에 안 잡힘.

## 실행 방법

사용자가 "스포티파이 형태로 블로그 자료 만들어줘 — [제목 또는 내용]" 라고 하면:

1. 홈페이지 `index.html`에서 해당 칼럼/가이드 내용을 읽어온다
2. `blog-assets/` 폴더에 `blog-[slug].html` 파일을 생성한다 (#thumb / #disc 섹션만 캡처 대상)
   - 디자인: 썸네일=스포티파이 다크(#121212, #1ed760), 면책=밝은 톤
3. `shot-blog.mjs`로 썸네일·면책 이미지 2장 캡처:
   ```bash
   node blog-assets/shot-blog.mjs blog-assets/blog-[slug].html
   ```
4. 본문은 `output-[slug]-본문.txt` 파일로 수기 정돈하여 작성 (네이버 복붙용)
   - 섹션 구분: `▌01. 제목` / 인용: `▪` / 경고: `⚠️`
5. 결과물(이미지 2 + 텍스트 1)을 사용자에게 보여주고 파일 위치 안내

## 출력 파일
```
blog-assets/
  output-[slug]-thumb.png   ← 네이버 대표 이미지
  output-[slug]-본문.txt     ← 본문 텍스트 (복붙 / 검색 색인용)
  output-[slug]-disc.png    ← 글 맨 아래 면책조항
```

## 네이버 블로그 업로드 순서
1. 글쓰기 → 대표 이미지: `thumb.png`
2. 본문에 `본문.txt` 내용 복사-붙여넣기
3. 맨 아래 이미지 삽입: `disc.png`
4. 발행
